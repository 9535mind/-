## 🎯 완성된 학습 흐름

### 학생 입장 (수강생 Journey)

```
1. 회원가입
   ↓
2. 로그인
   ↓
3. 강좌 둘러보기 (메인 페이지)
   ↓
4. 강좌 상세 페이지 확인
   - 강좌 정보
   - 가격 (할인 표시)
   - 차시 목록 (5분 표시)
   - 무료 미리보기 배지
   ↓
5. 수강 신청 (무료 강좌는 즉시, 유료는 결제)
   ↓
6. 내 강의실에서 학습하기 클릭
   ↓
7. 차시 영상 시청
   - 영상 재생
   - 자동 진도 체크
   - 다음 차시 이동
   ↓
8. 모든 차시 수료
   ↓
9. 수료증 발급 (준비 중)
   ↓
10. 강좌 리뷰 작성
   ↓
11. 다른 강좌 추천 받기 (준비 중)
```

### 관리자 입장

```
1. 관리자 로그인 (admin@mindstory.co.kr)
   ↓
2. 강좌 상세 페이지
   - "🛡️ 관리자 모드로 학습하기" 버튼 표시
   ↓
3. 모든 차시 접근 가능 (수강 신청 불필요)
   ↓
4. 대형 재생 버튼 (30% 크게)
   ↓
5. 관리자 대시보드
   - 학생 관리
   - 강좌 관리
   - 통계
```

### 2. 강좌 카드 디자인
- **카드 크기**: `p-6` (더 넓은 패딩)
- **차시 번호 뱃지**: `w-12 h-12` (더 큰 원형 뱃지)
- **차시 제목**: `font-bold text-lg` (굵은 글씨)
- **무료 미리보기 배지**: `bg-green-100 text-green-800 px-3 py-1 rounded-full` (녹색 배지)
- **호버 효과**: `hover:border-indigo-500 hover:shadow-lg` (테두리 색상 변경)

### 3. 진도율 시각화
- **프로그레스 바**: 색상 그라데이션 (0% → 빨강, 50% → 노랑, 100% → 녹색)
- **완료 차시 표시**: "완료 차시: X / Y"
- **시청 시간 표시**: "시청 시간: X분"

### 4. 반응형 디자인
- **모바일**: 1열 레이아웃
- **태블릿**: 2열 레이아웃

## 🐛 알려진 이슈

### 1. 팝업 404 에러
- **현상**: 팝업 API 호출 시 404 에러
- **영향**: 팝업 기능 작동 안 함 (기능적 영향 없음)
- **해결 방법**: 팝업 데이터 추가 필요

### 2. 썸네일 이미지 404
- **현상**: 네이버 포스트 썸네일 이미지 로드 실패
- **영향**: 썸네일 이미지가 표시되지 않음 (기능적 영향 없음)
- **해결 방법**: 로컬 이미지 업로드 권장

### 3. Tailwind CDN 경고
- **현상**: "cdn.tailwindcss.com should not be used in production" 경고
- **영향**: 없음 (개발 환경에서만 사용)
- **해결 방법**: 프로덕션 배포 시 PostCSS 또는 Tailwind CLI 사용
