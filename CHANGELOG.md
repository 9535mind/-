## Ver.2.2 - 회원 계정 관리 시스템 (2025-12-28)

### ✨ 새로운 기능

#### 회원 탈퇴 기능 (C안 방식)
- **탈퇴 사유 선택 필수**: 5가지 옵션 제공
  1. 사용하지 않는 서비스입니다
  2. 원하는 강의가 없습니다
  3. 다른 학습 플랫폼을 사용합니다
  4. 개인정보 보호를 위해
  5. 기타 (직접 입력)
- **조건부 차단**: 수강 중인 강의/결제 내역 있으면 탈퇴 불가
- **소프트 삭제**: 30일간 데이터 보관 후 완전 삭제
- **재가입 허용**: 동일 이메일로 재가입 가능
- **즉시 로그아웃**: 탈퇴 시 모든 세션 무효화

#### 내 정보 페이지 (`/my-profile`)
- 기본 정보 수정 (이름/전화번호/생년월일)
- 비밀번호 변경 (이메일 가입자만)
- 회원 탈퇴 섹션
- 소셜 로그인 사용자 구분

#### 카카오 소셜 로그인
- OAuth 2.0 기반 인증
- 자동 회원가입
- 프로필 정보 연동 (이메일, 닉네임)
